01 · Token

Color Tokens

Five primitives that drive every surface, border, and CTA on the sermon archive. Use semantic names (--bg-block) not primitives in components.

tokens/colors.css :root { --color-* }
CSS 
/* Primitives */
--color-white:        hsl(12, 33%, 97%);        /* #FAF3F0 */
--color-light-accent: hsl(50.16, 91.04%, 86.86%);/* #FCF2BF */
--color-accent:       hsl(36, 25%, 66%);          /* #BEAD93 */
--color-dark-accent:  hsl(27, 10%, 43%);          /* #7A6F65 */
--color-black:        hsl(120, 4%, 29%);           /* #474D47 */

/* Semantic — pick these, not the primitives */
--bg-site:    var(--color-white);
--bg-block:   var(--color-light-accent);
--bg-overlay: var(--color-dark-accent);
--text-primary: var(--color-black);

02 · Token

Typography & Callouts

Heading and body families plus the <em> callout swap — wrap a single word in <em> inside h1–h3 to switch fonts. Used on every sermon title and section heading.

tokens/typography.css h1 em, h2 em, h3 em, .sqsrte-small em
CSS 
--font-heading:           'aktiv-grotesk-extended', sans-serif;
--font-body:              'aktiv-grotesk-condensed', sans-serif;
--font-callout:           'span-semiboldcomp', sans-serif;
--font-secondary-callout: 'span-regularconditalic', sans-serif;

h1 em, h2 em, h3 em {
  font-family: var(--font-callout);
  font-style: normal;  /* em as typographic switch, not italic */
}

.sqsrte-small em {
  font-family: var(--font-secondary-callout);
  font-size: 28px;
  line-height: 28px;
  text-transform: none;
}

Hero Video Embed

Full-width 16:9 YouTube iframe on Dark Olive. Powered by the live/VOD switcher widget — falls back to the most recent video in the sermon playlist when no stream is live.

church-youtube-widget/squarespace-embed.html .radiant-hero / iframe wrapper
HTML 
<!-- Squarespace Code Block, placed at top of /messages or /sermon-archive -->
<div class="radiant-hero" style="background:#474D47">
  <div class="radiant-hero__frame">
    <iframe
      src="https://www.youtube.com/embed/VIDEO_ID?rel=0"
      title="Latest Sermon"
      allow="accelerometer; autoplay; clipboard-write;
             encrypted-media; gyroscope; picture-in-picture"
      allowfullscreen></iframe>
  </div>
  <a class="btn btn-primary radiant-hero__cta" href="#">
    Watch Latest Sermon
  </a>
</div>

08 · Layout

Next Steps Band

Full-width Soft Butter band with a 3-column grid: Salvation, Baptism, Find Your People. Each column has an icon, heading, description, and a ghost (tertiary) CTA.

layout pattern — section + button-tertiary .next-steps-band
HTML / CSS 
<section class="next-steps-band">
  <div class="next-steps-band__col">
    <div class="icon">✦</div>
    <h3>Salvation</h3>
    <p>Begin a relationship with Jesus today.</p>
    <a class="btn btn-tertiary" href="#">Learn More</a>
  </div>
  <!-- repeat for Baptism, Find Your People -->
</section>

.next-steps-band {
  background: var(--color-light-accent);
  padding:    var(--space-16) var(--space-12);
  display:    grid;
  grid-template-columns: repeat(3, 1fr);
  gap: var(--space-12);
}
